VCS using GitHub

Step 1:

1.1 Create an account on GitHub

1.2 Create a repository on GitHub

1.3 Provide Details of your repository

1.4 URL of your repository

Step 2:

2.1 Clone your repository by using clone command and providing URL of GitHub repository.

2.2 Then init your repository and add file to repository.

Step 3:

3.1 Now, run the commit command to save changes with a message to keep track of all changes made in repository.

3.2 Run the push command to upload/push changes to the server.

Step 4:

Now, I have made this change online in GitHub repository source code.

4.1 The git pull command is a combination of git fetch which fetches the recent commits in the local repository.

Step 5:

5.1 git branch command | is used to List all branches.

Step 6:

6.1 git checkout command is used to change position of head. While checkout -b command, makes a new branch and also moves the head to newly created branch.

Step 7:

7.1 git merge command is used to merge two branches.

Step 8:

8.1 git branch -D command is used to delete a branch.
